실험방법
1. 실험 전 준비
실험할 책상 위를 깨끗이 정리한다.
치아 모형, 치약 5종, 칫솔, 스포이드, 물, 자, 휴지, 기록지를 준비한다.
치아 모형 표면을 물로 가볍게 씻은 뒤 휴지나 키친타월로 완전히 말린다.
사용할 치약 5종에 이름표를 붙인다.
예: 어린이 치약, 일반 치약, 미백 치약, 잇몸 케어 치약, 젤 치약
2. 기준값 측정하기(치약을 바르지 않은 상태)

이 단계는 아주 중요하다. 나중에 비교할 기준이 된다.

깨끗하고 마른 치아 모형을 실험대 위에 놓는다.
스포이드로 물을 빨아들인다.
치아 모형 표면에서 약 3cm 높이에서 물방울 1방울을 떨어뜨린다.
물방울이 퍼진 뒤 바로 자로 가장 넓은 부분의 지름을 잰다.
기록지에 적는다.
같은 방법으로 5번 반복한다.
5번 측정한 값을 기록한 뒤 평균을 낸다.

예시 기록:
무처리 표면 1회, 2회, 3회, 4회, 5회, 평균

3. 치약 1종 실험하기

이제 치약 하나를 정해서 실험한다.

치약 1종을 선택한다.
치약을 약 0.5g 정도 짜서 칫솔에 묻힌다.
전자저울이 없으면 칫솔모 위에 짧고 얇게 1줄 정도로 맞춘다.
치아 모형 표면의 같은 위치를 30초 동안 같은 힘으로 문지른다.
문지른 뒤 물로 아주 가볍게 한 번 헹군다.
너무 세게 씻어서 치약 효과가 모두 사라지지 않도록 조심한다.
휴지나 키친타월로 물기를 제거하고 완전히 마를 때까지 3분 정도 기다린다.
다시 스포이드로 물을 빨아들인다.
같은 위치 위쪽 약 3cm 높이에서 물방울 1방울을 떨어뜨린다.
물방울이 퍼진 뒤 가장 넓은 부분의 지름을 자로 측정한다.
기록지에 적는다.
같은 치약으로 이 과정을 총 5번 반복한다.
5번 측정한 뒤 평균값을 계산한다.
4. 다른 치약들도 같은 방법으로 실험하기
첫 번째 치약 실험이 끝나면 치아 모형을 깨끗이 씻는다.
표면을 완전히 말린다.
두 번째 치약을 사용해 위와 똑같은 방법으로 실험한다.
나머지 치약도 모두 같은 방법으로 반복한다.
치약 종류만 바꾸고, 나머지 조건은 모두 같게 유지한다.

즉, 항상 같아야 하는 것:

치약 양
문지르는 시간
문지르는 힘
물방울 수
떨어뜨리는 높이
측정 방법
5. 결과 정리하기
각 치약별로 5번 측정한 값을 표로 정리한다.
각 치약의 평균값을 계산한다.
무처리 기준값과 비교한다.
평균값이 큰지 작은지 보고 어떤 치약에서 물방울이 더 많이 퍼졌는지 비교한다.
막대그래프로 정리하면 결과가 더 잘 보인다.

이건 하면 결과가 더 좋아져.

물방울은 항상 같은 스포이드로 떨어뜨린다
물은 항상 같은 종류의 물을 사용한다
물방울 크기가 비슷하도록 스포이드를 누르는 힘을 일정하게 한다
자를 옆에서 보지 말고 위에서 보고 잰다
가능하면 물방울을 위에서 사진 찍어 둔다
한 치약 실험이 끝날 때마다 표면을 깨끗이 씻고 말린다
보고서용으로 짧게 정리한 실험방법

보고서에 넣기 쉽게 짧은 버전도 줄게.

치아 모형을 깨끗이 씻고 건조시킨 뒤, 치약을 바르지 않은 상태에서 물방울 1방울을 떨어뜨려 퍼진 지름을 측정하였다. 이후 각 치약을 약 0.5g씩 사용하여 치아 모형 표면을 30초 동안 문지른 뒤 가볍게 헹구고 건조시켰다. 같은 높이에서 물방울 1방울을 떨어뜨려 퍼진 지름을 측정하였으며, 각 치약마다 5회씩 반복 실험하여 평균값을 구하였다.

추천 실험 조건 한 번에 정리
치약 종류: 5종
치약 사용량: 1회당 약 0.5g
문지르는 시간: 30초
건조 시간: 3분
물방울 수: 1방울
물방울 높이: 3cm
반복 횟수: 5회
