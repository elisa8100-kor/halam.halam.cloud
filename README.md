## 실험 준비물

- 치아 모형: 1~2개  
- 치약: 5종
- 칫솔: 1~2개  
- 스포이드: 1~2개  
- 물: 500mL ~ 1L  
- 자: 1개   
- 휴지 또는 키친타월: 10장 이상  
- 컵: 2개  
- 기록지: 2~5장  
- 스마트폰 카메라: 1대  
- 전자저울: 1개  
- 장갑: 1쌍  
- 격자 종이: 1~3장  

## 실험 기준

- 치약 사용량: 약 0.5g  
- 문지르는 시간: 30초  
- 물방울 수: 1방울  
- 물방울 떨어뜨리는 높이: 약 3cm  
- 반복 횟수: 5회
